Instalasi Sistem Operasi Proprietary Windows 10 di Virtual Box

  • Abdurrozzaaq Ashshiddiqi Zuhri
  • Nov 5, 2021

Langkah-langkah Instalasi Windows 10

1. Bagi kalian yang belum mempunyai aplikasi virtual box, maka download terlebih dahulu di sini

2. Setelah membuka website untuk mendownload virtual box, maka kalian pilih untuk windows, jika Komputer kalian selain Windows maka bisa kalian sesuaikan.

3. Setelah mendownload, kalian buka virtual box yang tadi telah di download untuk selanjutnya kita install, berikut tampilan awal saat kita membuka aplikasinya.

4. Selanjutnya pilih lokasi tempat virtual box akan disimpan.

5. Langkah selanjutnya kalian bisa pilih opsi dibuat shortcut atau tidak

6. Setelah berhasil menginstall virtual box, maka selanjutnya kita buka virtual boxnya

7. Klik tombol "New" untuk membuat virtual sistem operasi, kemudian isi dengan isian berikut

8. Isi virtual RAM yang akan kita buat pada virtual machine kita, kalian bisa mengikuti rekomendasi dari default virtual boxnya. Disini saya menggunakan 2048Mb. Setelah selesai klik "Next".

9. Kemudian create virtual hardisk

10. Memilih untuk membuat virtual hardisk, disini saya menggunakan VDI.

11. Tampilan setelah membuat virtual machine, selanjutnya kilk start pada virtual machine yang telah kita buat.

12. Memilih lokasi ISO dari system operasi yang akan kita install, disini saya menggunakan windows 10 64-bit 2016, kemudian pilih choose

13. Setelah selesai, maka klik "start" untuk memulai instalasi

14. Tampilan awal saat kita akan menginstall windows 10. Langsung klik "next" saja.

15. Klik Install Now

16. Memilih type windows yang akan kita install. Disini saya memilih Windows 10 Pro.

17. Menyetujui kebijakan dari microsoft, kemudian Next

18. Disini kita memilih custom dikarenakan kita belum memiliki system operasi didalam virtual yang telah kita buat, maka kita menginstall windows baru

19. Dalam bagian ini, kita memilih partisi yang akan kita jadikan tempat data dari system operasi ditaruh, kita memilih drive 0 lalu next

20. Menunggu proses install hingga selesai

21. Pilih use Express setting bagi pemula, karena langsung bisa diset secara default dari system operasi itu sendiri

22. Memberi nama pada user pc windows tersebut

23. Tampilan windows awal setelah install, Selamat mencoba!!!
